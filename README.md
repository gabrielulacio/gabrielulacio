## 👤 Sobre mí

<table>
<tr>
<td valign="top" width="60%">

**🚀 El Perfil**

Desarrollador Full-Stack de San Cristóbal, Venezuela. Me especializo en construir aplicaciones web end-to-end, aislando la complejidad del backend para entregar interfaces de usuario extremadamente rápidas e intuitivas.

**⚙️ El Enfoque**

Mi ecosistema principal es Vue.js interactuando con APIs RESTful en Python (FastAPI). Me obsesiona la correcta estructuración de datos, ya sea orquestando esquemas relacionales complejos o motores NoSQL.

**🎯 El Objetivo**

Busco integrarme a equipos de ingeniería como Desarrollador Full-Stack o Backend, aportando valor en productos transaccionales B2B o sistemas de alta concurrencia.

**⚡ Fuera del editor**

Cuando no estoy estructurando bases de datos, mi forma de desconectar es ver futbol, disfrutar de la F1 o jugar videojuegos tranquilos, estilo Minecraft. De vez en cuando, también le dedico tiempo a ajustar mi setup para mantener un entorno de trabajo lo más minimalista y funcional posible.
